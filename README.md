## Installation

1. Install `conda` and create an environment with `python` and `pip` installed.
2. Install the following dependencies
    - `qiskit` and `qiskit-aer`
    - `jupyter`
    - `matplotlib` and `pylatexenc`
3. Install the package by running `pip install -e .` from the project root.
